## Features
- **Project Management**
  - Create, edit, and delete projects
  - Kanban boards and task lists
  - Assign tasks to team members
  - Track progress with statuses and priorities

- **Task Management**
  - Create, edit, delete tasks
  - Set due dates and reminders
  - Attach files and links

- **Real-time Chat**
  - Team chat rooms for projects
  - Direct messages between users
  - Notifications for new messages
  - Emojis and file attachments

- **User Management**
  - Register, login, and profile management
  - Role-based permissions (Admin, Manager, Member)

- **Integration**
  - Google Calendar integration for task deadlines
  - Email notifications for updates

## Tech Stack
- **Frontend:** Flutter (Dart) 🎨  
- **Backend:** Dart + Node.js ⚡  
- **Database:** FireBase 🗄️  
- **Realtime:** Socket.IO for chat 💬  
- **Authentication:** JWT 🔐  
- **Deployment:** Docker 🐳, Heroku ☁️
